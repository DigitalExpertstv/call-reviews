# Call Reviews — Анализ диагностических звонков

Отчеты по диагностическим звонкам с клиентами. Анализ соответствия скрипту продаж.

## Скрипт диагностического звонка

Задачи звонка:
1. Провести диагностику клиента
2. Определить группу ЦА
3. Выявить потребности и контекст
4. Выявить возражения и боли
5. Сделать персонализированный оффер
6. В случае отказа — закрыть на следующий шаг
