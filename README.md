## Структура отчета

Каждый отчет содержит оценку по 7 блокам скрипта:
1. Диагностика (группа ЦА)
2. Потребности
3. Контекст
4. Возражения
5. Боли
6. Персонализированный оффер
7. Закрытие на следующий шаг

По каждому блоку: оценка, цитаты обеих сторон, что хорошо, что плохо, рекомендации.
